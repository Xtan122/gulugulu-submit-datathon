# Datathon 2026 Round 1

## Mô tả về cuộc thi

Đây là repository chứa code tham gia cuộc thi [Datathon 2026 Round 1](https://www.kaggle.com/competitions/datathon-2026-round-1) trên Kaggle.

## Cấu trúc Repository

- `visualization_eda.ipynb` - Notebook phân tích và trực quan hóa dữ liệu (EDA)
- `model.ipynb` - Notebook xây dựng và huấn luyện mô hình

## Cách chạy lại

Để tái lập kết quả, thực hiện các bước sau:

1. Truy cập vào link cuộc thi trên Kaggle: https://www.kaggle.com/competitions/datathon-2026-round-1
2. Vào mục **Code**
3. Chọn **New Notebook**
4. Upload notebook (`visualization_eda.ipynb` hoặc `model.ipynb`)
5. Chạy notebook để tái lập kết quả
